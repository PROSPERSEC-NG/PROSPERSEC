# OSI Model - Day 1

## Layer 1 - Physical
Cables, signals, hardware transmission.

## Layer 2 - Data Link
MAC addresses, switches, frames.

## Layer 3 - Network
IP addressing and routing.

## Layer 4 - Transport
TCP and UDP communication.

## Layer 5 - Session
Starts and manages communication sessions.

## Layer 6 - Presentation
Encryption, compression, formatting.

## Layer 7 - Application
User-facing applications and services.
